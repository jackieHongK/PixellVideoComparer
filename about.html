Capabilities

Everything experts need.
Nothing they don't.

🎯
Frame-accurate sync

requestVideoFrameCallback drift correction keeps all players locked to the master — even across 4 simultaneous streams with no perceptible lag.

4× streams
⚡
Instant frame stepping

WebCodecs + mp4box.js demux delivers sub-150ms response on arrow key steps for any length file — no more 1-second decode waits.

<150ms
📊
Per-player metric overlay

FPS, resolution, codec, dropped frames, bitrate, buffer — live sparkline graphs embedded in each video container. Toggle with one click.

🔁
Zero-stutter loop cache

Short clips (≤ 8s) are decoded once into GPU-resident ImageBitmaps. Every subsequent loop plays directly from cache — no re-decode, no stutter.

≤8s cache
🎬
Codec fallback

FFmpeg.wasm detects unsupported codecs and transcodes on-the-fly to H.264. No re-encoding pipeline. No desktop tools. Works everywhere.

🖼
Timeline thumbnails

Hover the scrubber to preview frames. Built instantly from cache for short clips, background-extracted for long ones. Speed over quality, always.
